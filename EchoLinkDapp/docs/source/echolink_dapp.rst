EKO Careers
--------------

EKO Careers is a blockchain based system that provides verified education, skill, and work experience information. Taking advantage of blockchain technology’s immutability and time stamp functionality, EKO Careers provides users with trusted information regarding a job candidate’s education, skill, and work experience. EKO Careers is built on the EKO Blockchain Platform.
